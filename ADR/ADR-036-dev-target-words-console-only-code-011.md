# ADR-036: Console-only debug-вывод target-слов без изменений UI

- Статус: accepted
- Дата: 2026-02-25
- Связанные задачи: [CODE]-[011]

## Контекст

Для задачи `[CODE]-[011]` требовалось добавить debug-доступ к списку target-слов уровня. Изначальный вариант с UI-панелью ухудшал UX и визуально конфликтовал с игровым экраном.

Критичное ограничение: production one-screen интерфейс должен оставаться неизменным, а debug-функция должна быть доступна только в dev-контуре.

## Решение

1. Отказаться от debug UI-панели целиком.
2. Реализовать dev-only вывод target-слов в консоль в `RenderMotion`:
   - префикс логов: `[dev][target-words]`;
   - payload: `levelId` + список `words[]` со статусом `found`.
3. Добавить дедупликацию по сигнатуре (`levelId + targetWords + foundTargets`), чтобы лог появлялся только при изменениях.
4. Удалить helper и тесты, относящиеся к panel UI, как неактуальные для выбранного решения.

## Последствия

- В dev-сборке target-слова доступны через консоль без визуального вмешательства в игровой экран.
- В production-сборке debug-логика не активируется.
- UI и layout релизного one-screen контракта сохранены без изменений.

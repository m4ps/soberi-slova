# ADR-009: Политика хранения временных init-артефактов

- Статус: accepted
- Дата: 2026-02-24
- Связанные задачи: [INIT]-[090]

## Контекст

В init-этапе накопились временные локальные артефакты (handoff-журнал агента и служебные каталоги
smoke/build), которые не являются source of truth проекта. Основной контур фиксации изменений уже
ведётся через `CHANGELOG.md`, `tasks/*.md` и ADR.

## Решение

1. Исключить временный `progress.md` из версионирования и удалить его из репозитория.
2. Зафиксировать правило в `.gitignore`, чтобы `progress.md` не попадал в коммиты.
3. Добавить воспроизводимую локальную приборку `npm run clean:init`, удаляющую
   `dist/`, `output/`, `.DS_Store` и `progress.md`.

## Последствия

- Репозиторий очищен от временных init-артефактов и содержит только целевые исходники и документацию.
- Любой участник может быстро привести рабочее дерево к чистому состоянию одной командой.
- История принятых решений и выполненных задач остаётся в проектных артефактах, а не во временных логах.
